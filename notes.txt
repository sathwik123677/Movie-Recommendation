========================================================================
                      CINEMATCH AI RECOMMENDATION ENGINE
========================================================================

CineMatch AI uses a hybrid filtering algorithm combining hard criteria
filters and weight-based metadata scoring to rank movies.

------------------------------------------------------------------------
PHASE 1: HARD FILTERING
------------------------------------------------------------------------
Before scoring, movies are filtered out based on basic user constraints:
- Language: Excludes movies not matching the selected language (unless "All").
- Rating: Excludes movies with IMDb ratings below the minimum threshold.
- Safety: If "Family-friendly" is chosen, "Adult" content ratings are excluded.
- Exclusions: Already-liked movies chosen in Step 2 are hidden from results.

------------------------------------------------------------------------
PHASE 2: WEIGHT-BASED SCORING
------------------------------------------------------------------------
For all remaining movies, a match score is calculated by adding points:

1. Favorite Genres:
   +4 points for each genre matching the selections in Step 1.

2. Taste Profile (Liked Movies Boost):
   If you select movies in Step 2, CineMatch extracts directors, cast members,
   and genres to calculate user-affinity boosts:
   - +3 points for movies sharing genres with liked movies.
   - +6 points for movies directed by a director of any liked movie.
   - +4 points for movies starring actors from any liked movie.

3. Vibe/Mood Matching:
   +5 points if the movie's mood catalog tags match the Step 3 vibe choice.

4. Runtime matching:
   +3 points if the duration matches the selected range:
   - Short: < 90 mins
   - Medium: 90 - 150 mins
   - Long: > 150 mins

5. Cast & Director Search Query:
   +10 points if the actor/director text query matches the cast or director.

6. IMDb Quality Boost:
   Adds the numeric IMDb rating directly (e.g. +8.2 points for an 8.2 rated movie)
   to prioritize overall quality.

------------------------------------------------------------------------
PHASE 3: RANKING & MATCH INSIGHTS
------------------------------------------------------------------------
1. Ranking: Movies are sorted by total score in descending order.
2. Insights: The specific reasons for point additions (e.g., "Similar genre
   to movies you liked") are collected and shown on the recommendation cards
   to provide clear, explainable feedback.
========================================================================


========================================================================
                      MOVIES.IPNB PROTOTYPE NOTEBOOK
========================================================================

The movies.ipynb file is a Jupyter Notebook used to prototype and evaluate
a machine learning Collaborative Filtering model on classic MovieLens data.

------------------------------------------------------------------------
1. THE SVD RECOMMENDATION ALGORITHM
------------------------------------------------------------------------
Unlike the lightweight hybrid rule-based engine in app.py, this notebook 
prototypes a Collaborative Filtering model using SVD (Singular Value 
Decomposition) from the surprise library:
- Dataset: Loads MovieLens 100k (ml-100k) rating files (u.data, 100,000 
  ratings from 943 users on 1,682 movies) and movie metadata (u.item).
- Model Training: Splits ratings data (80% train, 20% test) and fits the 
  matrix factorization SVD algorithm (n_factors=100).
- Evaluation: Computes the RMSE (Root Mean Squared Error) prediction 
  accuracy, which scores around 0.935 (predictions are off by less than 
  1 star on average).

------------------------------------------------------------------------
2. NOTEBOOK CODE CELLS STRUCTURE
------------------------------------------------------------------------
- Cell 1 & 2 (Data Loading): Reads MovieLens ratings and items pipe/tab 
  separated files into pandas DataFrames.
- Cell 3 (SVD training): Fits SVD and prints SVD model RMSE test metrics.
- Cell 4 (Top-N logic): Predicts ratings for all unseen movies for a 
  selected user, sorts them, and displays the top 10 movies.
- Cell 5 (Frontend parser): Formats predictions as JSON dict lists 
  (movie_id, title, predicted_rating).
- Cell 6 (Interactive Dropdown widgets): Uses ipywidgets to render an 
  interactive user selection dropdown to view recommendations.
========================================================================
