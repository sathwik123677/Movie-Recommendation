========================================================================
                   RECOMMENDATION ALGORITHMS SUMMARY
========================================================================

This project utilizes two distinct recommendation algorithms:

1. LIVE WEB APP ENGINE: Hybrid Content-Based & Profile Matching (app.py)
   - Real-time recommendation wizard.
   - Combines hard filters (language, safety, minimum rating) with 
     weighted metadata scoring.
   - Builds a personalized taste profile from checked liked movies.
   - Favors quality by adding raw IMDb ratings to the total score.

2. PROTOTYPE MACHINE LEARNING MODEL: Collaborative Filtering SVD (movies.ipynb)
   - Jupyter Notebook model prototype.
   - Uses Singular Value Decomposition (SVD) matrix factorization via 
     the Surprise ML library.
   - Trained on MovieLens 100k (100,000 ratings from 943 users on 1682 movies).
   - Predicts star ratings (1 to 5) for unrated movies to find top matches.

------------------------------------------------------------------------
LIVE WEB APP ENGINE WEIGHTS & FILTERING DETAILS (app.py)
------------------------------------------------------------------------
- Hard Filtering:
  * Excludes non-matching languages (unless "All").
  * Excludes ratings below user's selected threshold (e.g. < 7.0+).
  * Excludes "Adult" content ratings if "Family-friendly" is chosen.
  * Hides movies already checked in the Step 2 Liked checklist.
- Weight Scoring:
  * Favorite Genres: +4 points per matching selected genre.
  * Selected Vibe/Mood: +5 points if catalog tags match the mood.
  * Runtime Duration: +3 points if duration fits selected range.
  * Actor/Director Search: +10 points if search query matches cast/director.
  * IMDb Quality Boost: Adds raw rating directly (e.g. +8.3 score points).
- Liked Movies Boost (Taste Profile):
  * +6 points for movies directed by a director of a liked movie.
  * +4 points for movies starring cast members of a liked movie.
  * +3 points for movies sharing genres with liked movies.

------------------------------------------------------------------------
PROTOTYPE SVD MODEL DETAILS (movies.ipynb)
------------------------------------------------------------------------
- Matrix Factorization SVD:
  * Factorizes the User-Item ratings matrix into user and item latent 
    factors (using n_factors=100).
  * Evaluated on test data split (80% train, 20% test).
  * Achieves RMSE (Root Mean Squared Error) accuracy of ~0.935.
- Code Cells Layout:
  * Cells 1 & 2: Reads ratings (u.data) and items (u.item) into DataFrames.
  * Cell 3: Trains SVD model and prints test RMSE.
  * Cell 4: Predicts ratings for unseen movies and prints top 10 list.
  * Cell 5: Formats top-N outputs as frontend-ready JSON lists.
  * Cell 6: Renders interactive widgets dropdown to view SVD matches.
========================================================================
